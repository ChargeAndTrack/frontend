# Frontend

ChargeAndTrack frontend for ASW and SPE projects.

## How to use

Follow this steps in order to run the frontend:

- clone the repository or download the latest release;
- go to the root directory (`/frontend`);
- run the following commands:

```bash
npm run build
npm run preview
```
